Portfolio Webpage

This portfolio website spans several HTML pages.

It uses PrismJS for colorful code blocks.

This website also uses Vectary for displaying 3D models within a page.

One drawback I found while making this website is that Chrome doesn't allow a page to import any external files. I wanted to import each extra HTML page into the main "page" <div>, as this would save copying and pasting the same <header> and <footer>.Chrome doesn't allow this even if the file is in the same directory. This also means that the File-Highlight plugin for PrismJS does not work. This is why each page of the website has to be its own HTML file.
