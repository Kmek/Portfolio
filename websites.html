Portfolio Webpage

This portfolio website spans several HTML pages. PrismJS is used on several pages for colorful code blocks. This website also uses Vectary on the grabcad.html page for easily displaying 3D models within a page.

Using PrismJS

The files for PrismJS can be found in the /prism/ folder. Then the CSS and JavaScript is just included into the webpage. To make a code block, enclose the code for the block inside a set of <pre> and <code> components. Depending on the plugins added, a label can be displayed by setting the data-label in the <pre> component. Line numbers, bracket highlighting, and rainbow brackets can be added with the line-numbers, match-braces, and rainbow-braces classes to the <code> component.

<head>
	<link href="prism/prism.css" rel="stylesheet"/>
</head>
<body>

	<pre data-label="Example Code Block">
		<code class="language-html match-braces rainbow-braces line-numbers">
			<!-- Code to display goes here -->
		</code>
	</pre>

	<script src="prism/prism.js"></script>
</body>

One drawback I found while making this portfolio website is that Chrome doesn't allow a page to import any external files. I wanted to import each extra HTML page into the main "page" <div>, as this would save copying and pasting the same header and footer.Chrome doesn't allow this even if the file is in the same directory. This also means that the File-Highlight plugin for PrismJS does not work. This is why each page of the website has to be its own HTML file.

Portfolio Page Layout

The basic layout for each HTML file starts with a <div> with the "header" id. Inside of this is a <h1> with the "title" id, which links back to the main index.html page. The next <div>, "pageBackground", holds the patterned paper-like background separate from the page contents. The page contents reside in their own <div>, "page". The last <div> is the "footer", which holds the bottom left "footerText" note and the bottom right "githubBtn" button. This button links to the GitHub page for this portfolio's repository.

<body>
	<div id="header">
		<a href="./index.html"><h1 id="title">Kate Keefe's Portfolioh1></a>
	</div>

	<div id="pageBackground">
		<div id="page">

			<!-- Contents of the page go here -->

		</div>
	</div>

	<div id="footer">
		<span id="footerText">Portfolio & website made by Kate Keefe</span>
		<a href="https://github.com/Kmek/Portfolio" target="_blank"><button id="githubBtn">View on GitHub</button></a>
	</div>
	
	<script src="prism/prism.js"></script>
</body>
